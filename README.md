# Sistema de Gerenciamento de Karaokê

### Instalação e Execução

- Clone o projeto:

```bash
git clone https://github.com/karaoke-manager-project/front.git
```

- Entre no repositório:

```bash
cd front
```

- Instale as dependências:

```bash
npm install
```

- Crie um .env com as mesmas variáveis do .env.example:

```.env
VITE_FRONT_URL=
VITE_PORT=
```

- Rode o projeto:

```bash
npm run dev
```
