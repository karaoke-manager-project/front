# Sistema de Gerenciamento de Karaokê

### Instalação e Execução

- Clone o projeto:

```bash
git clone https://github.com/karaoke-manager-project/front.git
```

- Entre no repositório:

```bash
cd front
```

- Instale as dependências:

```bash
npm install
```

- Rode o projeto:

```bash
npm run dev
```
